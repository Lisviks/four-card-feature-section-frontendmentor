### What I learned

`box-shadow: h-offset v-offset blur spread color`, blur, spread and color are optional.

```html
<div class="card">
  <h1>Box Shadow</h1>
  <p>Box shadow example</p>
</div>
```

```css
.card {
  width: 200px;
  height: 200px;
  box-shadow: 0 10px 10px 10px grey;
}
```
